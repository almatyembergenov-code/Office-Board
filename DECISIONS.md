# Принятые решения

## Единый дашборд без переходов
Office Board реализуется как один общий полноэкранный дашборд без отдельных страниц и меню.

Причина: экран должен постоянно быть открыт на втором мониторе и быстро показывать ситуацию в офисе.

## Только для команды из трёх человек
Проект создаётся для Алмата, Захара и Ильяса. Не планируется как система для всей Академии.

## Квартальный календарь
Сохраняем формат: прошлый месяц / текущий месяц / следующий месяц.

Причина: команда привыкла видеть прошлые действия, текущую занятость и ближайшие планы.

## Цветовая логика
Сохраняем привычные цвета:
- розовый — тренинги;
- зелёный — тесты;
- оранжевый — АП / адаптация;
- жёлтый — курсы / встречи;
- красный — выходной;
- синий — отпуск.
