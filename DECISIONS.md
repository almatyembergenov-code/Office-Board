# Решения проекта

## Единый экран вместо многостраничного сайта
Office Board реализуется как один полноэкранный дашборд без переходов между страницами.

## Google Calendar как источник данных
Редактирование событий остается в Google Calendar. Office Board только отображает события.

## ICS через GitHub Secrets
Секретная ICS-ссылка не вставляется в `index.html` и не хранится в репозитории. GitHub Actions забирает события и сохраняет безопасный файл `data/events.json`.
