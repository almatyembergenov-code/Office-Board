# Office Board

Office Board — единый полноэкранный дашборд для команды из трёх человек: Алмат, Захар, Ильяс.

Цель: за 3–5 секунд показать, что происходит в офисе сейчас, сегодня и в ближайшие месяцы.

## Версия 0.1

- квартальный календарь: прошлый / текущий / следующий месяц;
- карточки статусов сотрудников;
- цветовые категории событий;
- панель выбранного дня;
- крупные часы и дата;
- блок «Важно на сегодня»;
- поле заметок.

Пока данные лежат в `data.js`. Позже подключим Google Calendar.
